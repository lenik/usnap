
多继承的问题
--------------------------------------------------------



> 名字冲突时，实现等效。

	存储结构-------------------------------------
		&pC 12ff60 ->
			pC 4216a0 ->
				Vtbl 414020 ->
					C::f1 401005 ->
						C::f1 I1/impl 4011A0

			pC+4 ->
				Vtbl 41401C ->
					C::f1 40100A ->
						C::f1 I2/impl 401260 ->
							jmp C::f1 401005

					C::C 40100F

			pC+8 ->
				m_1 01234567
				m_2 89ABCDEF


	可以看出，当一个类的两个基类具有相同名字时，

		CBase1	CBase2
		  |       |
		  \---+---/
		      |
		      V
		  CDerived

		此时CDerived只能给出一种实现，且这种实现对CBase1,CBase2等效。
		（这通过 CDerived/CBase2::func jmp--> CDerived/CBase1::func 来完成）
