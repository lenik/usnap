

	obj F1(obj x) {
		return x;
	}

	obj F2(obj y) {
		return y;
	}

	main() {
		obj i;
		F2(F1(i));
	}

	1. F1(i) 将生成i的副本, 设i_d1, F1返回i_d1, 当调用F1的语句结束后i_d1自动销毁.
	2. 调用F1的是F2()语句, F2(i_d1)将生成i_d1的副本i_d1_d1, 当调用F2的语句结束后i_d1_d1自动销毁
	3. 调用F2的是main的第3行, "F2(F1(i))"

	所以,

		生成	i			// main函数中obj i引起
		生成	i_d1		// 编译器生成内部拷贝	(程序无法知道)
		生成	i_d1_d1		// 编译器生成内部拷贝	(程序无法知道)
		销毁	i_d1_d1		// 调用副本的销毁程序
		销毁	i_d1		// 调用副本的销毁程序
		销毁	i			// main }引起
