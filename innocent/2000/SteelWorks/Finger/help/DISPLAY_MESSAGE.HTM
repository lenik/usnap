DISPLAY MESSAGE

[服务端]通知客户需要立即显示一条消息。

格式：

DATA TEXT FING #DISPLAY MESSAGE#消息...

功能：

当服务器需要向客户发送一条消息并要求客户响应时使用。
除了使用DISPLAY MESSAGE外还可以使用DATA TEXT SCRP向客户发送显示脚本。

返回：

成功：OK

失败：NOT A CLIENT

注：

不同于SERVER MESSAGE，虽然这两个都是服务端的消息，但SERVER MESSAGE用于传送消息，而DISPLAY MESSAGE仅用于需要客户响应的场合。

参见：

SERVER MESSAGE
DATA TEXT SCRP
